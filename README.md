# ESP32s3-Ili9341-Touch
Connect ESP32S3 to ILI9341 TFT With Touch

Use the provided User_Setup.h for the TFT_eSPI Library
put the file in \Arduino\libraries\TFT_eSPI
is the configuration for the ESP32s3
